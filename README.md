# index-html-gen

Node module that generates a index.html template.

>  <!doctype html>
>  <html>
>  <head>
>      <title></title>
>      <link rel="stylesheet" type="text/css" href="filename.css" />
>      <script src="filename.js"></script>
>  </head>
>  <body>
>  </body>
>  </html

## Usage

```sh
$ node path/to/index-html-gen optionalDirectory
```
